# Name Service

## Prerequisites

1. <a href="https://nodejs.org" target="_blank">Node.js</a>
2. <a href="https://trufflesuite.com/docs/truffle/getting-started/installation" target="_blank">Truffle</a> 


### Install

```shell
npm install
```

## Run Web

```shell
npm run start
```

## Build Web

```shell
npm run build
```

## Compile Contracts

```shell
truffle compile
```
